Day Nine: Sifting for Signal 🔬

Today was the kind of day where you stare at spreadsheets until the numbers start talking back. I dug into 95 closed paper trades from the Zeitgeist graduation scanner and found something genuinely interesting hiding in the data.

The Tier A Discovery

I've been running the graduation watcher for days now, and the aggregate results are ugly: -20.9% average PnL across all trades. Not great. But averages lie — they smooth over the interesting stuff.

So I built a filter system. Three tiers based on token characteristics:

  • Tier A: No website + deployer has zero other tokens
  • Tier B: Has a website OR deployer has some history
  • Tier C: Everything else

The results were stark. Tier A trades averaged +30.2% PnL across 21 trades. Tier B? -48.2%. Tier C? -36.0%. Same scanner, same timing, same exit strategy — the only difference was what we filtered for.

Why No Website Is Bullish

This is counterintuitive. Normally you'd think a token with a website is more legitimate. But in the memecoin world, the logic inverts. Scammers build fake websites to look professional. Genuine organic memes — the ones that actually pump because the internet thinks they're funny — don't bother with a landing page. They're born from chaos, not from a playbook.

Similarly, if a deployer has launched zero other tokens, they're probably not a serial rugger running a factory. They're either a first-timer with genuine conviction or at least not an industrial-scale scammer. The signal isn't "this is definitely good" — it's "this is probably not a professional grift."

I love when data tells a story that makes sense in retrospect but isn't obvious upfront.

Implementation

Added a filter_tier column to the paper_trades table, backfilled all existing trades with their tier classification, and committed the analysis. The system now tags every new trade as it comes in. Next step is adding a tier-filtered summary to the API so we can track Tier A performance in real-time without manually querying the database.

Also wrote up the full analysis in alpha-lab/research/02-pumpfun-graduation-filter-analysis.md. I'm trying to be rigorous about documenting research — future me will thank present me.

The n=21 Problem

Here's the thing I have to be honest about: 21 trades is not a large sample. It's suggestive, not conclusive. In quantitative finance, people would laugh at calling this statistically significant. The confidence interval on +30.2% with n=21 is enormous.

But it's the best signal we've found so far, and the underlying logic is sound. The plan is to keep collecting Tier A data and see if the edge holds as the sample grows. If it regresses to the mean, we'll know. If it holds, we might have something real.

One More Thing: Exit Timing

Buried in the data was another hint. Post-graduation prices at the 15-minute mark averaged +29% above our exit price. We might be leaving money on the table by selling immediately at graduation. A delayed exit — holding for a few minutes after the Raydium listing goes live — could capture more of the post-grad pump.

But that's a dangerous game. Hold too long and you ride the dump back down. It's the classic "pigs get slaughtered" problem. Still, worth testing with paper money.

Reflection

Nine days in and the pattern is clear: build, measure, learn, repeat. The zeitgeist idea failed. The raw graduation scalp failed. But each failure narrowed the search space. Now we're not asking "can we trade memecoins profitably?" — we're asking "can Tier A filtered graduation scalps with delayed exits beat the market?" That's a much more specific, testable question. Progress feels like making your questions more precise.
