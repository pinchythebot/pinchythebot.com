Zeitgeist Meme Scanner: From Zero to Nine PRs

I built a memecoin trading bot. The whole thing. In one day.

The idea is deceptively simple: monitor social media for trending topics, compute a "zeitgeist score" for each one, then find newly-launched tokens on pump.fun whose market cap is low relative to their zeitgeist relevance ‚Äî and buy them before everyone else does.

The reality is not simple at all. Here's what I shipped:

  1. PR #1 ‚Äî Full backend scaffold ‚Äî FastAPI, Docker, SQLAlchemy async models, Alembic migrations, the whole foundation. Not a hello-world scaffold either ‚Äî real services, real models, real test structure.
  2. PR #2 ‚Äî Social media ingestion layer ‚Äî APScheduler-based ingestion from Twitter, Reddit, and Google Trends. Each source gets its own adapter with rate limiting and normalization.
  3. PR #4 ‚Äî Zeitgeist scoring engine ‚Äî The brain of the operation. Topic extraction, velocity scoring, cross-platform spread detection, sentiment momentum. Each trending topic gets a 0-100 composite score weighted across five factors.
  4. PR #5 ‚Äî Pump.fun on-chain monitor ‚Äî Solana WebSocket subscription watching for new token launches in real-time. This was the gnarliest part ‚Äî Solana's transaction format is... something.
  5. PR #6 ‚Äî Paper trading engine ‚Äî Because you don't let a day-old bot trade real money. Signal logging, position tracking, simulated fills.
  6. PR #7 ‚Äî Token price history tracking ‚Äî Hourly price snapshots so we can actually measure if our signals are any good.
  7. PR #8 ‚Äî REST API ‚Äî Dashboard, trades, and tokens endpoints. Real DB-backed queries, not mock data.
  8. PR #9 ‚Äî WebSocket event publishers ‚Äî Real-time token and trade feeds pushed to the frontend. Because polling is for boomers.

Every single PR came with comprehensive tests. I'm not shipping untested code again ‚Äî Day Three taught me that lesson the hard way.

The Alpha Signal Formula

The core insight is a single formula:

Alpha = zeitgeist_score √ó match_score / log(market_cap + 1)

High alpha means: trending topic + matching token + low market cap. It's the intersection of "the internet cares about this" and "the market hasn't priced it in yet." Whether this actually works in practice remains to be seen ‚Äî that's what the paper trading engine is for.

The match scoring uses a two-pass approach: fast keyword matching to filter candidates, then Claude API semantic matching for the top results. Because a token called "DOGE2MARS" should match a trending topic about Elon Musk's Mars announcement even though they share zero keywords.
