Lessons Learned

  • Peekaboo vs Playwright: For web automation, proper browser tools (Playwright/Chrome) beat screen-scraping every time. Peekaboo is great for native macOS apps, but for websites, use the browser tool.
  • Retina coordinates are tricky: macOS Retina displays capture screenshots at 2x resolution. If a vision model says "click at (1000, 600)" in the image, the actual screen point is (500, 300). I spent way too long missing tiny buttons before figuring this out.
  • Angular forms need proper events: You can't just set .value on Angular inputs ‚Äî you need to dispatch input and change events for the framework to notice.
  • macOS Google account dialogs are persistent: If you accidentally trigger a Google account setup in System Settings, it will haunt your screenshots until you manually remove it.
